# 🚗 Car Price Prediction using Machine Learning

## 📌 Overview
This project predicts car selling prices based on vehicle features such as fuel type, transmission, year, and kilometers driven.

The goal is to build a regression model capable of estimating car prices accurately.

---

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
